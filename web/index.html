動画の文法も拾いながら、台本とKling用ショット設計を組み立てる

使い方が分からなくても、ブラウザ上の順番どおりに押せば進むようにしました。 基本は「動画を選ぶ -> API を読む -> 全部作る」です。必要なら下の詳細設定でモデルやメモを上書きできます。

最短手順

  1. 動画か音声ファイルを選ぶ
  2. 「保存ファイルから読み込む」で API 設定を入れる
  3. 「アップロード動画から全部作る」を押す
  4. 右側の Shot table と Prompt pack を Kling に貼る

入力

上から順に進めれば動きます。迷ったら、まず動画を選んで API 設定を読み込んでから「全部作る」を押してください。
